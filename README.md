## Getting Started

1. Clone the repository: `git clone https://github.com/RovicAngelo/WMS.git`
2. Navigate to the project directory: `cd WMS`
3. Compile and run the application:

```bash
$ javac Main.java
$ java src/com/lanuza/wms/Main
```
